# X-Ray Body-Part Classifier — Demo

Interactive demo for [emon5122/xray-bodypart-classifier](https://huggingface.co/emon5122/xray-bodypart-classifier):
a ConvNeXt-Tiny model that predicts the imaged anatomy from a plain radiograph across **33 classes**.

Upload an X-ray and see the top-5 predicted body parts with confidence.

> ⚠️ **Suggestion/demo only — NOT for clinical or diagnostic use.**
